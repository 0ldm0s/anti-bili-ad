# B站推荐过滤

给B站不可屏蔽的推荐内容添加过滤功能，支持按类型或UP主屏蔽。

## 功能

- **一键屏蔽**：鼠标悬停在推荐卡片上，点击屏蔽按钮即可过滤
- **按类型屏蔽**：可屏蔽整类内容（番剧、直播、纪录片、综艺、电视剧、电影、国创、漫画、课堂）
- **UP主屏蔽**：跨类型屏蔽特定UP主的所有推广内容
- **本地存储**：所有过滤数据保存在本地，不上传服务器
- **简单管理**：可视化界面管理黑名单

## 使用方法

1. 安装扩展后，访问B站首页
2. 找到直播、番剧等不可屏蔽的推荐卡片
3. 鼠标悬停在卡片上，右上角会出现一个屏蔽按钮（X图标）
4. 点击按钮即可将该内容加入黑名单
5. 下次访问时，该内容会被自动隐藏

## 管理黑名单

- 点击浏览器工具栏中的扩展图标
- 查看已过滤的内容列表
- 可以单独移除某条过滤规则
- 也可以一键清空所有黑名单
- 支持"按类型屏蔽"功能，一键屏蔽整类内容

## 支持的内容类型

番剧、直播、纪录片、综艺、电视剧、电影、国创、漫画、课堂

## 技术实现

- 使用 MutationObserver 监听页面动态加载的内容
- 通过 `.floor-single-card` 选择器识别不可屏蔽的卡片
- 使用 chrome.storage.local API 持久化存储黑名单
- UP主推广使用 `user:UP主名` 格式（跨类型屏蔽）
- 内容推广使用 `type:名称` 格式（如"番剧:物理魔法使马修"）
